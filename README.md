# Page View Time Series Visualizer

This project is a **time series visualizer** for the FreeCodeCamp Forum page views. It provides three types of visualizations using **Python**, **Pandas**, **Matplotlib**, and **Seaborn**:

1. **Line Plot** – Shows daily page views over time.  
2. **Bar Plot** – Displays average monthly page views per year.  
3. **Box Plot** – Shows year-wise and month-wise distributions to highlight trends and seasonality.
